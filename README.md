# MOVIMIENTO-DE-DIVS-Y-TIEMPOS-DE-ANIMACI-N
Animación web desarrollada con HTML y CSS que simula el movimiento de dos elementos (abejas) dentro de un contenedor con fondo. Se utilizan @keyframes para definir trayectorias, tiempos de animación y efectos de desplazamiento y rotación, ejecutándose de forma continua e infinita.
